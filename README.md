# breast-cancer-prediction-ml-app
Breast Cancer Prediction ML Project: Machine Learning model using Logistic Regression to predict benign vs malignant tumors from the Breast Cancer Wisconsin dataset. Includes data preprocessing, model evaluation, and a Streamlit app for interactive predictions.

Breast Cancer Prediction Machine Learning App

This project combines Machine Learning and an interactive web interface to predict whether a breast tumor is benign or malignant using the Breast Cancer Wisconsin (Diagnostic) dataset. The dataset includes features like radius, texture, perimeter, area, smoothness, and other measurements of cell nuclei, which are critical for accurate diagnosis.

The workflow begins with data exploration and preprocessing, including handling missing values, scaling features, and preparing the dataset for modeling. A Logistic Regression model is trained to classify tumors, chosen for its interpretability and strong performance in binary classification. Model evaluation uses metrics such as accuracy, precision, recall, and F1-score, ensuring reliable predictions.

To make the model accessible, a Streamlit web application is developed. The app allows users to input medical feature values and receive instant predictions of tumor type. Screenshots of the app demonstrate a clean and interactive interface, where users can clearly see the input fields and prediction results. The model and preprocessing objects are serialized using pickle, enabling smooth integration with the app.

This project showcases a full end-to-end machine learning solution: from data handling and model training to deployment and user interaction. By combining ML with a user-friendly Streamlit interface, it demonstrates how AI can provide practical, real-time assistance in healthcare decision-making. The project also highlights skills in Python, pandas, NumPy, scikit-learn, and Streamlit, making it a strong addition to any portfolio.
